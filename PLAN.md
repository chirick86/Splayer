# ОБЩИЙ ПЛАН

## доработка поиска

при поиске конкретного исполнителя/альбома по uri выводит список треков данного альбома/артиста. Вроде бы спотифай отдает сразу все треки альбома при запросе конкретного альбома.

## доработка плейлиста

### добавление в плейлист (--add)

* добавление в плейлист по запросу (запрос отправляется в обработчик поисковика)
можно запросить uri, url или название трека/исполнителя/альбома.
* при добавлении артиста, добавляются все его треки в плейлист (сколько отдает в поисковике)
* при добавлении альбома, добавляется весь альбом
* при текстовом запросе добавляется только первая найденная песня.
Для этого будет использоваться субаргумент --add
* добавляться все будет в текущий плейлист. Если текущего плейлиста нет, будем выводить сообщение о том что не выбран плейлист

### default playlist (--default)

* Выбор дефолтного плейлиста, который будет использоваться при добавлении треков в плейлист без указания конкретного плейлиста. Если не выбран дефолтный плейлист, выводить сообщение о том что не выбран плейлист.
* доработать вывод плейлистов. Сейчас там указывается current playlist, но не указывается дефолтный плейлист. Сделать так, чтобы выводился дефолтный плейлист так же помечался (default)
* решить куда добавлять треки если есть и дефолтный и текущий плейлист. Возможно интерактивный выбор с вопросом и вводом ответа c/d
c - current playlist
d - default playlist
